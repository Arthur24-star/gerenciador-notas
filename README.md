# Sistema de Gerenciamento de Notas

## Descrição do Projeto

Este projeto consiste em um sistema simples de gerenciamento de notas de estudantes desenvolvido em Python. O sistema permite armazenar informações acadêmicas, calcular médias, verificar aprovação dos estudantes e gerar relatórios organizados no terminal.

O objetivo principal da aplicação é aplicar conceitos fundamentais de programação, como estruturas de dados, funções, modularização, documentação e testes.

---

## Funcionalidades

- Cadastro de estudantes e notas
- Cálculo automático de médias
- Verificação de aprovação ou reprovação
- Geração de relatório acadêmico
- Estrutura modularizada em funções
- Testes básicos de validação

---

## Tecnologias Utilizadas

- Python 3

---

## Estrutura do Projeto

- `gerenciador_notas.py` → arquivo principal do sistema
- `test_notas.py` → arquivo contendo os testes do sistema

---

## Como Executar o Sistema

### 1. Abrir o terminal

Abra o terminal do VS Code ou do sistema operacional.
